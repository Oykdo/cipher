┌──────────────────────────────────────────────────────────────────────────┐
│                        📋 PREMIÈRE UTILISATION                           │
└──────────────────────────────────────────────────────────────────────────┘

Lors du premier lancement :

1. Les dépendances seront installées automatiquement (2-5 minutes)
   
2. L'application démarrera avec 3 services :
   - Backend (serveur) sur le port 4000
   - Frontend (interface) sur le port 5173
   - Fenêtre Electron (application de bureau)

3. Créez votre compte :
   - Cliquez sur "Créer un compte"
   - Choisissez "Standard" (phrase de 12 ou 24 mots)
   - NOTEZ BIEN votre phrase de récupération sur papier !
   - Choisissez un nom d'utilisateur


┌──────────────────────────────────────────────────────────────────────────┐
│                    ✨ FONCTIONNALITÉS À TESTER                           │
└──────────────────────────────────────────────────────────────────────────┘

▸ Messagerie sécurisée
  → Tous les messages sont chiffrés de bout en bout (E2E)
  → Le serveur ne peut jamais lire vos messages

▸ Time-Lock (Messages temporisés) ⏰
  → Cliquez sur l'icône horloge lors de l'envoi
  → Le message sera verrouillé jusqu'à l'heure choisie
  → Impossible de le lire avant !

▸ Burn After Reading (Auto-destruction) 🔥
  → Cliquez sur l'icône flamme lors de l'envoi
  → Le message s'autodétruira après lecture
  → Choix : 5s, 30s, 1min, 5min

▸ Authentification avancée 🎲
  → Standard : Phrase BIP-39 (12 ou 24 mots)
  → DiceKey : Clé physique Dice-Key (6 mots)

┌──────────────────────────────────────────────────────────────────────────┐
│                       🐛 PROBLÈMES COURANTS                              │
└──────────────────────────────────────────────────────────────────────────┘

❌ "Node.js n'est pas installé"
   → Installez Node.js v18+ depuis https://nodejs.org/

❌ L'application ne démarre pas
   → Vérifiez que les ports 4000 et 5173 sont libres
   → Fermez toute autre application utilisant ces ports

❌ "Port already in use"
   → Un autre programme utilise le port
   → Fermez toutes les applications et réessayez

❌ La fenêtre ne s'ouvre pas
   → Vérifiez les logs dans le terminal
   → Essayez d'ouvrir manuellement : http://localhost:5173

❌ Erreur lors de l'installation
   → Supprimez le dossier "node_modules"
   → Relancez cipher.bat
